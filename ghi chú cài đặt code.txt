 **các folder
 + views: chứa giao diện trang ejs(html)
 + public: chứa css,js,image 
 + configs: chứa file kết nối data lên SQL connect.BD(sửa 'database' để chuyển data khác nếu cần)
 + controller: chứa các chức năng để liên kết các file,Thêm,sửa,xóa
 + route: chứa các đường dẫn của các trang
